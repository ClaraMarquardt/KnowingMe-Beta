We want to help you understand how you communicate.

To do this, we (i) collect your email address and gather some basic information from your Gmail Profile (full name + profile photo) and (ii) pull the text and metadata of your sent and received emails over a specified period from your Gmail account. Whether you are using the Desktop or Web app, we are committed to safeguarding the data we collect.


Desktop App
All the data that we collect is stored and analyzed locally on your computer.
No data is collected, analyzed or retained by us in any other location.



Web App
All the data that we collect is stored and analyzed on a web server hosted by Amazon Web Servers (AWS). No data is collected, analyzed or retained by us in any other location. The data is secured with a user-session-specific password and accessible only to our team's CTO. Any data that is transferred to the server is automatically deleted within 24 hours as part of a daily server reboot.

In either case you can delete all your data by following these steps to properly terminate the app: (i) Log out of the app, confirming your intent to reset the app when prompted; (ii) Close this browser window; (iii) (Desktop App Only) Terminate the Docker container in which the app is being executed; and (iv) (Desktop App Only) Delete the user-designated data folder.

We realize that you're placing a large trust in our service.
If you have concerns that aren't covered here, please contact us.
